    Authentication

    Make sure to replace Client ID with your client ID.

    Brick issues each client a client ID and client secret. The client application needs to encrypt the client ID using client secret using JWT as an access token.

    You can register a new Brick API key at our website https://brick-api-site.herokuapp.com/

    Brick API expects access token in each API requests to the server in a header that looks like the following:

    accessToken: xxxxxxxxxxxxxxxxxxxxxxxxxx


    You must replace client ID and client secret with your client ID and client secret.

    Brick Connect

    Introduction

    Brick connect is a module to connect to the banks’ API. We provide a javascript code snippet to the client to open the Brick connect either as a page or popup.
    Client app will see a list of banks in the Brick Connect page. Once the bank icon is clicked, users will be redirected to the bank’s OAuth2 page to authorize using their bank crendentials. Once it’s done, we will redirect back to the callback URL.

    • Please keep the client secret carefully and do not share it
    • Brick security server will maintain the access token with various banks for the data retrieval

    Errors

    This error section is stored in a separate file in includes/_errors.md. Whiteboard allows you to optionally separate out your docs into many files…just save them to the includes folder and add them to the top of your index.md‘s frontmatter. Files are included in the order listed.

    The Kittn API uses the following error codes:

    Error Code Meaning
    400 Bad Request – Your request sucks
    401 Unauthorized – Your API key is wrong
    403 Forbidden – The kitten requested is hidden for administrators only
    404 Not Found – The specified kitten could not be found
    405 Method Not Allowed – You tried to access a kitten with an invalid method
    406 Not Acceptable – You requested a format that isn’t json
    410 Gone – The kitten requested has been removed from our servers
    418 I’m a teapot
    429 Too Many Requests – You’re requesting too many kittens! Slow down!
    500 Internal Server Error – We had a problem with our server. Try again later.
    503 Service Unavailable – We’re temporarily offline for maintenance. Please try again later.
